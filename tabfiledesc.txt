HEADER:
0x00-0x02:	"TAB"
0x03:		tab file version
0x04-0x1F:	0x00

INFO:
0x20-0x3F:	song title
0x40-0x5F:	song author

TUNING:
0x60-0x61:	song tuning id
0x62-0x6F:	song tuning settings

MEASURES:
0x70-0x71:	measures number
0x72-0x7F:	0x00
0x80:		beginning of list of measures

measure entry:
	0x20 bytes for title
	0x01 byte for metronome change (0 if none)
	0x01 byte for time signature change (0 if none)
	0x04 bytes pointer to the list of notes
	0x0A null bytes
